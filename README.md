

## Atari Breakout

![Atari Breakout](printscreens/printscreenBreakOut.png "$(Atari breakout)")

### Description

classical Atari Breakout game



### Controls

Keyboard:
 - A go left
 - D go right
 - P to pause
 - ESC to exit
